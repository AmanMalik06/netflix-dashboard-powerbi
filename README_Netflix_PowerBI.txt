NETFLIX DATA ANALYSIS DASHBOARD (POWER BI)

Overview
--------
This project presents an interactive Power BI dashboard built on the Netflix Movies and TV Shows dataset.
It focuses on uncovering patterns, trends, and distribution of Netflix content across years, countries, and genres.

What This Dashboard Shows
-------------------------
- Total number of titles available on Netflix
- Distribution of Movies vs TV Shows
- Content spread across countries and genres
- Trends based on release year and date added
- Interactive filtering for exploratory analysis

Tools Used
----------
- Power BI Desktop
- Power Query for data cleaning and transformation
- Basic DAX measures for analysis

Data Preparation
----------------
- Cleaned raw Netflix dataset
- Handled missing values and date fields
- Standardized categorical attributes such as country and genre

Key Takeaways
-------------
- Netflix content has grown rapidly over recent years
- Movies form a larger share compared to TV Shows
- Content diversity varies significantly by region and genre

How to Use
----------
Open the .pbix file in Power BI Desktop to interact with the dashboard.
